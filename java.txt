8. Khi khai báo dependency có 1 số đặc điểm như: có version, không có version, sử dụng starter
Vậy mấy đặc điểm đó là như thế nào, phân tích rõ
sử dụng starter: khi sử dụng starter là sử dụng starter parent của spring framework phát triển
project mình sử dụng là project con -> thừa kế project parent của spring framework
parent có sẵn thư viện, có sẵn cấu hình theo phiên bản

có version khai báo khi parent project chưa có dependency này
=> cần phải khai báo version cho dependency

không có version khai báo khi mà parent project có dependency này
=> không cần phải khai báo version

version của dependency phụ thuộc vào version của starter parent

có version
không có version
